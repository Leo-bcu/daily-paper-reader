## 摘要
大型视觉语言模型 (LVLMs) 在语义理解方面表现出色，但在细粒度空间定位 (spatial grounding) 方面面临挑战，因为模型必须在不产生空间解释的情况下隐式推断复杂的几何结构。我们提出了 Perceptio，这是一种具有 2D 和 3D 空间推理能力的感知增强型 LVLM，通过在自回归序列中直接生成的显式语义分割 Token 和深度 Token 来实现。具体而言，我们 (i) 从强大的单目教师模型中蒸馏出 VQVAE 深度码本，将密集深度标记化为紧凑序列；(ii) 在 LLM 中集成基于 SAM2 的语义分割 Token 和 VQ-VAE 深度 Token，使模型先输出空间 Token 再进行回答。为了稳定深度 Token 的生成，我们引入了新型复合深度 Token 目标函数（标记、Token 和计数损失）以及一种用于可微重建的软合并 (soft-merging) 技术。我们在不同数据集上采用了多任务协同训练策略，让模型学习感知 Token 以应对多个下游任务。基于 InternVL，Perceptio 在各项基准测试中均达到了最先进的性能：在 RefCOCO/+/g 上将指代词分割 (referring expression segmentation) 提高了 +0.8/+1.4/+1.1 cIoU，在 HardBLINK 空间理解准确率上提高了 10.3%，在 MMBench 准确率上提高了 1.0%，证明了显式的空间思维链 (spatial chain-of-thought) 能够实质性地增强 LVLM 的空间定位能力。

## Abstract
Large Vision Language Models (LVLMs) excel at semantic understanding but struggle with fine grained spatial grounding, as the model must implicitly infer complex geometry without ever producing a spatial interpretation. We present Perceptio, a perception enhanced LVLM with 2D and 3D spatial reasoning abilities, enabled via explicit semantic segmentation tokens and depth tokens generated directly within the autoregressive sequence. Concretely, we (i) distill a VQVAE depth codebook from a strong monocular teacher to tokenize dense depth into compact sequences, and (ii) integrate SAM2 based semantic segmentation tokens and VQ-VAE depth tokens inside the LLM so the model first emits spatial tokens and then answers. To stabilize depth token generation, we introduce novel composite depth-token objectives (marker, token, and count losses) and a soft-merging technique for differentiable reconstruction. We adopt a multi-task co-training strategy across diverse datasets, letting the model learn perception tokens to tackle multiple downstream tasks. Building on InternVL, Perceptio achieves state-of-the-art performance across benchmarks: improving referring expression segmentation by +0.8/+1.4/+1.1 cIoU on RefCOCO/+/g HardBLINK spatial understanding accuracy by 10.3%, and MMBench accuracy by 1.0%, demonstrating that explicit spatial chain-of-thought materially strengthens spatial grounding in LVLMs.

现代大型视觉语言模型（LVLM，如 InternVL、Qwen-VL）虽然在语义理解和对话方面表现出色，但在**细粒度空间定位和 3D 空间推理**（如深度、距离、比例）方面仍然薄弱。研究发现，即使是强大的模型在处理人类“眨眼间”就能解决的空间任务（如 HardBLINK 基准）时，表现也接近随机猜测。
**核心动机**：现有的 LVLM 缺乏显式的空间线索，通常是隐式地从池化特征中推断几何关系。本论文旨在通过在自回归序列中引入显式的 2D 语义分割和 3D 深度 Token，构建一种“空间思维链”（Spatial Chain-of-Thought），从而增强模型的空间智能。

### 2. 方法论：核心思想与关键技术
Perceptio 的核心思想是让模型在生成最终文本答案之前，先生成代表空间信息的 Token。
*   **感知增强生成序列**：模型输出格式定义为：`[seg tokens] [depth tokens] [text tokens]`。这种顺序强制模型先解释场景的空间结构，再进行逻辑回答。
*   **2D 语义分割**：集成 SAM2 编码器。模型预测一个特殊的 `[seg]` Token，触发 SAM2 解码器生成与查询相关的掩码（Mask）。
*   **3D 深度 Token 化**：
    *   使用 **VQ-VAE** 从专家模型（Depth Anything V2）中蒸馏深度信息，构建一个包含 128 个码本条目的深度码本。
    *   将连续深度图离散化为 100 个 Token 的序列。
*   **关键算法与损失函数**：
    *   **复合深度 Token 损失**：包括标记损失（Marker Loss，定位起始/结束）、Token 损失（Token Loss，数值准确性）和计数损失（Count Loss，序列长度一致性）。
    *   **软深度重建（Soft Depth Reconstruction）**：为了实现端到端训练，采用软合并技术，将离散 Token 的预测概率分布与码本嵌入加权求和，使深度重建过程可微。
*   **多任务协同训练**：在统一的指令微调框架下，同时优化文本生成、分割重建和深度预测。

### 6. 主要结论与发现

### 7. 优点
*   **统一性**：首次在单个自回归序列中联合优化了 2D 语义和 3D 几何感知。
*   **可微性创新**：通过软合并技术解决了离散 Token 预测与连续图像重建之间的梯度断裂问题。

*   **优化张力（Optimization Tension）**：消融实验发现，加入深度 Token 后，通用 VQA 指标（如 MMBench）相比“无深度版本”有轻微下降（约 0.4%），说明深度生成任务与纯文本任务之间存在一定的竞争关系。

*   **教师模型依赖**：模型性能受限于教师模型（SAM2 和 Depth Anything V2）的质量，教师模型的错误会传播给学生模型。
